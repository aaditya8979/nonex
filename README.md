## 🚀 &nbsp;Local Deployment

### Prerequisites

```bash
node  >= 18.17.0
npm   >= 9.0.0
```

### Step 1 — Clone

```bash
git clone https://github.com/YOUR_GITHUB_USERNAME/nonex.git
cd nonex
```

### Step 2 — Install

```bash
npm install
```

### Step 3 — Environment

Create `.env.local` in the project root:

```env
# ─────────────────────────────────────────
# APEX VALUE ENGINE — Environment Variables
# ─────────────────────────────────────────

# Required: OpenRouter API key
# Get yours at → https://openrouter.ai/keys
OPENROUTER_API_KEY=sk-or-xxxxxxxxxxxxxxxxxxxxxxxx

# Optional: Override default model
# OPENROUTER_MODEL=anthropic/claude-3-haiku
```

### Step 4 — Run

```bash
npm run dev
```

Open **[http://localhost:3000](http://localhost:3000)** to launch the terminal.

```bash
# Production build
npm run build && npm start

# Type checking
npm run type-check

# Linting
npm run lint
```
